Detailed Description

3rd person camera controller.

Child Class of Parent Camera_2 for Patriots of the Past. Inherits initial camera settings from Camera_2. Uses mouse input to orbit camera in a 3D space and mouse wheel for zoom, input works with joysticks. Camera follows character if no mouse input is detected. Includes methods to handle player occlusion and object/environment clipping
